얀(yarn)
-자바스크립트 패키지 매니저
- npm 보다 성능(속도)와 보안성이 우수함

설치명령어
npm install -g yarn

버전 확인
yarn -v


리액트 설치 주의사항
1.상위 또는 현재 폴더 명이 "한글"로 되어있는 경우 -한글 불가
2. node.js 버전이 LTS 가 아닌경우 오류

리액트 앱 설치
npx create-react-app [프로젝트명]
예: npx create-react-app 01_practice

	정상 설치 완료시
	cd 01_my-app
	npm start
	를 입력하시오 라고 나옴
	
$cd 프로젝트명
$npm start
입력 시 리엑트 실행 완료





최신 버전의 reactstrap을 노드 모듈에 추가하는 방법
// --save 옵션은 node_module 안에 저장하겠다. 라는 말임.
npm install --save [패키지명]
npm install --save [패키지명]@[버전]

예
npm install --save reactstrap
npm install --save reactstrap@8
npm install --save bootstrap
npm install --save bootstrap@4.6

yarn add [패키지명]
